# Portfolio Tibo TESSIER

Portfolio personnel de Tibo Tessier, etudiant en BUT Informatique. Site vitrine full stack avec animations, carrousel de projets, statistiques GitHub et WakaTime, et sections parcours/competences.

## Apercu
- Accueil avec hero, CTA et effet Matrix
- Projets avec modal et carrousel d'images
- Competences en plusieurs categories
- Parcours (formation et experiences)
- Statistiques dynamiques (GitHub, WakaTime)
- Contact et reseaux

## Stack
- HTML5, CSS3, JavaScript (ES6)
- AOS 2.3.1, GSAP 3.12.2
- Font Awesome 6.4.0, Google Fonts (Montserrat)
- Vercel Serverless Functions pour /api

## Usage

Ce depot est un portfolio personnel. Il n'est pas prevu pour etre redeploye par des tiers.

Si vous souhaitez vous en inspirer, merci de ne pas reutiliser les contenus, l'identite visuelle ou les informations personnelles.

## Notes

Les routes `/api` utilisent des fonctions serverless Vercel pour les statistiques.
