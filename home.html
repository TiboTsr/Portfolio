Générateur de QR Code en Python

Projet réalisé dans le cadre de mon portfolio, le générateur de QR Code en Python offre une solution simple et efficace pour créer des codes QR personnalisés.

L'objectif initial de ce projet était de développer un outil polyvalent permettant la génération de QR codes avec des fonctionnalités avancées, le tout en utilisant le langage de programmation Python.

Cet outil propose les fonctionnalités suivantes :

  • • Personnalisation avancée des QR codes (taille, version, couleur, etc.)
  • • Prise en charge de divers types de données (texte, URL, etc.)
  • • Téléchargement facile des QR codes générés
  • • Interface utilisateur en ligne de commande
  • • Enregistrement automatique dans le dossier de téléchargement

Pour télécharger la dernière version de l'outil, rendez-vous sur la page des versions disponibles.

Nouveautés de la version 7
  • • Langue sélectionnable (français ou anglais)
  • • Niveaux de difficulté pour une personnalisation adaptée à tous
  • • Validation des couleurs améliorée
  • • Enregistrement automatique optimisé
  • • Corrections et améliorations diverses

Python QR Code Generator

Project carried out as part of my portfolio, the Python QR Code generator offers a simple and effective solution for creating customized QR codes.

The initial objective of this project was to develop a versatile tool allowing the generation of QR codes with advanced features, all using the Python programming language.

This tool offers the following features:

  • • Advanced customization of QR codes (size, version, color, etc.)
  • • Support for various types of data (text, URL, etc.)
  • • Easy download of generated QR codes
  • • Command-line user interface
  • • Automatic saving in the download folder

To download the latest version of the tool, go to the page of available versions.

New features in version 7
  • • Selectable language (French or English)
  • • Difficulty levels for customized personalization
  • • Improved color validation
  • • Optimized automatic saving
  • • Various corrections and improvements
